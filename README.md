# Capstone 3: Power BI Sales Analysis for EmporiUm

## Project Overview
This Power BI report analyzes four years of sales data for **EmporiUm**, a virtual student bookstore resulting from the merger of college bookstore franchises and an online retailer. The analysis focuses on sales trends, category performance, geographic distribution, and top-selling general audience books for one sales region.

The goal is to provide actionable insights to the regional sales team (director, territory managers, and store managers) to support better business decision-making.


## Report Structure
- **Page 1**: Executive Dashboard
- **Page 2**: Category Analysis + Top Books

Each page includes clear, descriptive headers.

## Technologies Used
- **Power BI Desktop** (Data modeling, DAX, visualizations)
- **Power Query** (Data cleaning and transformation)
- **Excel + Text files** (Source data)
- **Power BI Service** (Published report)

## Data Sources
- Large Excel file containing sales transactions (2022–2025)
- Separate text file with book titles and author information for general audience books

## Key Insights
- Strong year-over-year sales growth with seasonal peaks
-  Study Guides is the top-performing category in the region
- Maryland drives the majority of regional revenue
- Top authors and books identified for inventory and marketing focus
